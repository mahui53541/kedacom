# kedacomAssessment
科达实习考核项目

##考核目标
**掌握基于Spring Boot的微服务开发**

##考核项目
**实现电商应用基础服务，包括用户服务，商品服务，订单服务**

**功能包括用户注册，用户鉴权，商品列表，商品详情，下单，查看订单列表，订单详情**

##评分细则:
  1. 使用Spring Boot, Spring JPA实现底层服务的CRUD 40分
  2. 单元测试，API测试 10分
  3. 符合RESTful API规范 10分
  4. 实现webservice接口查询订单  10分
  5. 实现服务注册发现组件 10分
  6. 实现服务网关组件 10分
  7. 实现服务的高可用 10分
